There are 3 python files each contains a different script.

here is what each does:

1)Joke.py uses joke api, calls the api or a get request and in return get a response shows that response in terminal. I used time library to have a lag or sleep of 5 sec
after it shows the first line just for curiosity.

2)wheather_CLI.py uses a wheather api but in this api it requires longitude, latitude and an api key so i stored the api key in an env file loaded it in my program
and used it also latitude and longitude will be input in terminal.

3)AI_impact_analysis.py downloads a data set from kaggle by using kaggle api then uses pandas library to perform some basic statistics.

both the api keys were held in env file for safety
